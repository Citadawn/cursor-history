# Suggested Commands

## Development
```bash
npm run build        # Build TypeScript
npm run dev          # Watch mode
npm test             # Run tests (vitest)
npm run test:watch   # Test watch mode
npm run lint         # ESLint
npm run lint:fix     # ESLint with auto-fix
npm run format       # Prettier format
npm run typecheck    # TypeScript type check
```

## Running the CLI
```bash
node dist/cli/index.js list           # List sessions
node dist/cli/index.js show 1         # Show session 1
node dist/cli/index.js search "query" # Search sessions
```

## Git (Linux)
```bash
git status
git add .
git commit -m "message"
git push
```

## File System (Linux)
```bash
ls -la
cd /path
grep -r "pattern" .
find . -name "*.ts"
```
